Gugusay 4.0 使用说明
====================

首次使用：
---------
1. 确保已有 data/ 文件夹（包含 SR.db 数据库）
2. 确保已有 media/ 文件夹（存放图片/视频）
3. 如果没有，请从之前版本复制或新建空文件夹

数据结构：
---------
- data/SR.db      : 推文数据库文件（必须）
- media/          : 媒体文件存放目录（可选）

版本说明：
---------
- 本版本为绿色便携版，无需安装
- data 和 media 文件夹可与旧版本共享
- 直接双击 twitter.exe 启动

在线更新：
---------
点击"时光机" -> "设置" -> "检查更新"可在线更新数据库和媒体文件
